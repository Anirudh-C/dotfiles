# Setup

## Software
* Arch
* i3 window manager
* Polybar
* tmux
* [vim](https://github.com/anirudh-c/fiddler-vim)
* [emacs(spacemacs)](https://github.com/anirudh-c/spacemacs)
* zsh

## Configs
### OS
    |
    |--> ~/.xinitrc
    |--> ~/.Xresources
    |--> ~/.config/i3/config
    |--> ~/.config/polybar/config
    |--> Monitor
         |
         |--> /etc/X11/xorg.conf.d/10-monitor.conf
    |--> Keyboard
         |
         |--> ~/.xmodmap

### Work
    |
    |--> Terminal
         |
         |--> ~/.config/alacritty
    |--> Vim
         |
         |--> ~/.vim
         |--> ~/.vimrc
         |--> ~/.viminfo
         |--> ~/vimwiki
    |--> Emacs(Spacemacs)
         |
         |--> ~/.emacs.d
         |--> ~/.spacemacs
    |--> ~/.tmux.conf
    |--> Shell
         |
         |--> ~/.zshrc
         |--> ~/.oh-my-zsh
